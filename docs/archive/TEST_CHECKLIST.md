### المواعيد
- [ ] حجز موعد → فحص avl + تعارض + منفذ + طلب إتمام
- [ ] عرض عدد المواعيد النشطة
- [ ] منع حجز على عرض فيه طلب إتمام
- [ ] دورة التراشق (5 جولات)
- [ ] عرض "جاهز بأي وقت" (avl=any) → تظهر أيام الأسبوع السبعة بدوام 09:00–21:00 (من app_config.appt) وليس النص "any"
- [ ] عرض بـ avl فارغة → السيرفر يرفض بـ NO_AVAILABILITY (وليس تخطي الفحص)
- [ ] قاعدة الساعة: موعد نشط 10:00 → 10:30 مظللة بالواجهة ومرفوضة سيرفرياً (TIME_CONFLICT_ON_OFFER)، وأقرب حجز مسموح 11:00
- [ ] قاعدة الساعة على المشرف: مشرف لديه موعد 10:00 لا يُسند له موعد قبل 11:00 (ينتقل للمشرف التالي الأقل حمولة)
- [ ] لا مشرف متاح → لا يُنشأ الموعد + رسالة تتضمن أقرب موعد مقترح + إشعار في notifications (tp=2)
- [ ] الـ Sheet لا يعمل Overflow مع فترات طويلة (تمرير داخلي + حد أقصى 85% من الشاشة)
- [ ] رسائل الأخطاء المميزة: DUPLICATE_APPOINTMENT / TIME_CONFLICT_ON_OFFER / NO_SUPERVISOR_AVAILABLE / NO_AVAILABILITY
- [ ] التراشق: اقتراح وقت بديل ضمن أقل من ساعة من موعد نشط آخر على نفس العرض → مرفوض برسالة فارق الساعة
- [ ] التراشق: اقتراح وقت في الماضي → مرفوض (INVALID_APPOINTMENT_TIME)
- [ ] التراشق بلا مشرف متاح → فشل مُدار برسالة + اقتراح بديل + إشعار (بدون خطأ خام)
- [ ] التراشق الناجح يعيد تعيين supervisor_uid للمشرف المتاح في الوقت الجديد
