### شاشة إضافة عرض احترافية (2026-06-13)

- فصل حقول العقار عن السيارة — كل نوع يرى فقط حقوله المناسبة.
- عقار: مناطق (السويداء/صلخد/شهبا + إدخال حر) + مساحة + إكساء (5 خيارات) + طابق + اتجاه (8 اتجاهات) + ملاحظات قانونية + خريطة.
- سيارة: لوحة + محافظة (14 محافظة) + ماركة + موديل + سنة + لون + كم + وقود (5 خيارات) + ناقل (3 خيارات) + سند سيارة (3 أنواع) + نوع نمرة (3 أنواع).
- عنوان تلقائي ذكي: عقار "تصنيف في منطقة" / سيارة "ماركة موديل سنة".
- المواعيد المتاحة (stepAvl) تظهر للعقار فقط.
- `_totalSteps` ديناميكي + إعادة step عند تبديل النوع + loc يحفظ محافظة السيارة.

### تفاصيل العرض وبطاقة العرض (2026-06-13)

- عرض كل حقول specs بأيقونات حسب نوع العرض (عقار/سيارة).
- بطاقة العرض: عقار يعرض موقع+مساحة / سيارة يعرض ماركة+سنة+كم.
- رقم العرض (#) يظهر بالبطاقة وبتفاصيل العرض.

### تحسينات عامة (2026-06-13)

- سبلاش: "المكتب العقاري الالكتروني" أولاً + "السويداء" تحتها.
- إشعارات: "تم نشر العرض الخاص بك" + رقم العرض بالنص.
- رقم عرض تسلسلي: عمود `offer_number` + sequence + trigger تلقائي.
- نقاط: تُمنح فقط عند مشاركة فعلية (ShareResultStatus.success) + عرض عدد النقاط.
- checkbox نشر على السوشال (مفعّل افتراضياً).
- فلتر العروض المطابقة بتفاصيل الطلب (سعر من-إلى + موقع).
- زر حجز موعد مباشرة من بطاقة العرض المطابق.

### إصلاح حجز المواعيد ودورة حياة العرض (2026-06-12)

- `book_appointment_internal` أُعيد كتابتها بالكامل — 10 فحوصات:
  1. العرض منشور (sts=2)
  2. منع حجز عرضك
  3. وقت بالمستقبل
  4. فحص طلب إتمام معلق على العرض
  5. فحص avl (يوم + فترة زمنية بتوقيت دمشق)
  6. فحص تعارض العقار (نفس الوقت)
  7. منع تكرار نفس المستخدم
  8. اختيار منفذ (role=3) متاح
  9. إنشاء مع supervisor_uid
  10. إرجاع عدد المواعيد النشطة
- `process_completion_request`: عند الموافقة يحوّل `offers.sts=5` + يلغي كل المواعيد الأخرى + يرسل إشعار لأصحابها.
- `create_deal_internal`: `role>=5` + يحوّل `offers.sts=5`.
- `complete_deal_internal`: `role>=5` + يحوّل `offers.sts=6` + يلغي مواعيد متبقية.
- `admin_force_appointment_internal`: `role>=4`.
- `admin_handle_report_internal`: `role>=4`.
- `create_request_internal`: إعفاء `role>=4`.

### تدقيق عميق (2026-06-12)
