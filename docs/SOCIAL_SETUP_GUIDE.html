المرحلة 1 إنشاء صفحة فيس بوك تجارية

⏱️ الوقت المقدر: 10 دقائق

1 فتح Meta Business Suite
  1. افتح المتصفح على الكمبيوتر
  2. ادخل على: business.facebook.com
  3. سجّل دخول بحسابك الشخصي على فيس بوك
  4. إذا سألك "Create a business portfolio" — اضغط Create Account
2 إنشاء Business Portfolio
  1. Business portfolio name: المكتب العقاري الالكتروني
  2. Your full name: اسمك الحقيقي
  3. Business email: إيميلك (مو شرط نفس إيميل فيس بوك)
  4. اضغط Submit
3 إنشاء صفحة فيس بوك جديدة
  1. من القائمة اليسار اضغط Settings ⚙️
  2. اضغط Business assets → Pages
  3. اضغط Add assets → Facebook Page → Create a new Facebook Page
  4. املأ البيانات:
    • Page name: المكتب العقاري الالكتروني
    • Category: اكتب Real Estate واختار Real Estate Agent
    • Bio: منصتك العقارية الذكية — بيع، إيجار، وتصوير عقاري احترافي 🏠
  5. اضغط Create Page
  6. أضف صورة شعار التطبيق كصورة الملف (Profile picture)
  7. أضف صورة غلاف (Cover photo) — ممكن أي صورة عقارية مؤقتة
4 إعدادات الصفحة المهمة
  1. روح للصفحة الجديدة واضغط Settings → Page Access
  2. تأكد إنك Admin على الصفحة ✅
  3. اضغط Settings → Privacy
  4. تأكد إنو الصفحة Published (منشورة) مش Unpublished
مرحلة 1 مكتملة! عندك صفحة فيس بوك تجارية باسم المكتب العقاري الالكتروني

المرحلة 2 إنشاء حساب انستغرام تجاري

⏱️ الوقت المقدر: 15 دقيقة

1 إنشاء حساب انستغرام جديد
  1. حمّل تطبيق Instagram على موبايلك
  2. اضغط Create new account
  3. اسم المستخدم (Username): sweeda.real_estate أو sweeda.realestate أو elc.real.estate
  4. كلمة السر: اختار كلمة سر قوية واحفظها!
  5. الاسم الكامل: المكتب العقاري الالكتروني
2 التحويل لحساب تجاري (Business)
  1. افتح البروفايل → اضغط ≡ (القائمة) أعلى اليمين
  2. اضغط Settings and privacy → Account type and tools
  3. اضغط Switch to professional account
  4. اختار Business
  5. اختار التصنيف: Real Estate
  6. اربط الحساب بصفحة فيس بوك اللي أنشأناها بالمرحلة 1
    بيطلب منك تسجيل دخول فيس بوك → اختار صفحة "المكتب العقاري الالكتروني"
  7. اضغط Done
3 إكمال البروفايل
  1. أضف صورة الملف (نفس شعار التطبيق)
  2. أضف Bio: 🏠 منصتك العقارية الذكية | بيع • إيجار • تصوير
  3. أضف رابط: رابط التطبيق أو الموقع
تأكد من الربط: من الكمبيوتر ادخل على business.facebook.com → Settings → Business assets → Instagram accounts. المفروض تشوف حساب الانستغرام مربوط.

المرحلة 4 الحصول على التوكنز

⏱️ الوقت المقدر: 30 دقيقة

🎯 هذه المرحلة الأكثر تعقيداً — اتبع خطوة بخطوة

كيف تعمل التوكنز؟

Short-lived User Token (ساعة واحدة) │ ▼ نستبدله بـ Long-lived User Token (60 يوم) │ ▼ نستبدله بـ Long-lived Page Token (لا ينتهي أبداً! ✅)
1 الحصول على Short-lived User Token
  1. افتح: developers.facebook.com/tools/explorer/
  2. اختار التطبيق: Sweeda Real Estate Publisher (من القائمة يمين فوق)
  3. اضغط User or Page → User Token
  4. اضغط Generate Access Token
  5. بيظهر قائمة الصلاحيات — اختار كل الصلاحيات الخمسة بالجدول فوق
  6. وافق على الأذونات
  7. انسخ التوكن اللي ظهر ← هذا هو Short-lived User Token
2 استبداله بـ Long-lived User Token

افتح Terminal أو موجه الأوامر ونفّذ:

curl -X GET \ "https://graph.facebook.com/v21.0/oauth/access_token\ ?grant_type=fb_exchange_token\ &client_id=APP_ID\ &client_secret=APP_SECRET\ &fb_exchange_token=SHORT_LIVED_TOKEN"

استبدل:

  • APP_ID ← من المرحلة 3 خطوة 5
  • APP_SECRET ← من المرحلة 3 خطوة 5
  • SHORT_LIVED_TOKEN ← من الخطوة 1 فوق

النتيجة رح تكون JSON فيها access_token جديد — هذا هو Long-lived User Token (60 يوم)

انسخه واحفظه!
3 الحصول على Page ID
curl -X GET \ "https://graph.facebook.com/v21.0/me/accounts?fields=id,name\ &access_token=LONG_LIVED_USER_TOKEN"

النتيجة:

{ "data": [ { "id": "123456789012345", "name": "المكتب العقاري الالكتروني" } ] }
انسخ الـ id — هذا هو Facebook Page ID
4 الحصول على Long-lived Page Token (لا ينتهي! 🎉)
curl -X GET \ "https://graph.facebook.com/v21.0/PAGE_ID?fields=access_token\ &access_token=LONG_LIVED_USER_TOKEN"

النتيجة:

{ "access_token": "EAAxxxxxxxxxxxxxxxxxxxxxx", "id": "123456789012345" }
هذا هو Long-lived Page Token — لا ينتهي أبداً! 🔑 احفظه بمكان آمن جداً
5 الحصول على Instagram Business Account ID
curl -X GET \ "https://graph.facebook.com/v21.0/PAGE_ID?fields=instagram_business_account\ &access_token=PAGE_TOKEN"

النتيجة:

{ "instagram_business_account": { "id": "17841400000000000" }, "id": "123456789012345" }
انسخ الـ id داخل instagram_business_account — هذا هو Instagram Business Account ID

المرحلة 5 اختبار التوكنز

قبل ما تبعت البيانات، لازم تتأكد إنو التوكنز شغالة!

اختبار فيس بوك — إنشاء منشور تجريبي

curl -X POST \ "https://graph.facebook.com/v21.0/PAGE_ID/feed\ ?message=🏠 تجربة النشر التلقائي من المكتب العقاري الالكتروني\ &access_token=PAGE_TOKEN"

إذا رجع:

{"id": "123456789012345_987654321"}
فيس بوك شغال! احذف المنشور التجريبي من الصفحة يدوياً

اختبار انستغرام — إنشاء منشور صورة تجريبي

انستغرام لازم رابط صورة عامة (URL)، مش رفع مباشر
// الخطوة 1: إنشاء حاوية curl -X POST \ "https://graph.facebook.com/v21.0/IG_BUSINESS_ID/media\ ?image_url=https://upload.wikimedia.org/wikipedia/commons/thumb/a/a7/Camponotus_flavomarginatus_ant.jpg/320px-Camponotus_flavomarginatus_ant.jpg\ &caption=🏠 تجربة النشر التلقائي #عقارات\ &access_token=PAGE_TOKEN"

بيرجع container ID. انتظر 10-15 ثانية ثم:

// الخطوة 2: نشر الحاوية curl -X POST \ "https://graph.facebook.com/v21.0/IG_BUSINESS_ID/media_publish\ ?creation_id=CONTAINER_ID\ &access_token=PAGE_TOKEN"
إذا ظهر المنشور على الانستغرام — كل شيء شغال!

❓ أسئلة شائعة

هل التوكن رح ينتهي؟

الـ Page Token لا ينتهي أبداً طالما التطبيق فعال على ميتا. بس الـ App Secret ممكن يتغير إذا غيّرت كلمة سر فيس بوك، فلازم تحدّثو.

هل أحتاج App Review؟

للـ testing مع Admin فقط — لا. بس لو بدك مستخدمين آخرين يستخدمو التطبيق — نعم، لازم تقدم على App Review من الداشبورد. بيآخذ 3-10 أيام.

انستغرام بينشر نص بس؟

لا — انستغرام لازم صورة على الأقل. النشر التلقائي رح يستخدم أول صورة من العرض. إذا العرض ما فيه صور → ينشر على فيس بوك بس.

هل أقدر أشوف المنشورات اللي اتنشرت؟

اي، من Meta Business Suite → Planner أو Content تقدر تشوف كل المنشورات المجدولة والمنشورة.
